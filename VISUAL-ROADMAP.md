## FLUJO DE INGRESOS

```
POR CADA 1,000 VISITANTES A TU LANDING:

Landing → Quiz         (65% = 650 personas)
  │
  ├─ Quiz → Email      (61% = ~400 personas)
  │
  ├─ Email → Resultado (87% = ~350 personas)
  │
  ├─ Resultado → Checkout (10% = ~35 personas)
  │
  └─ Checkout → Pagada (95% = ~33 personas)

┌─ Visitantes:            1,000
├─ Leads qualified:         400
├─ Checkout initiated:       35
├─ Conversión final:         33  (3.3%)
└─ Revenue (€297):      €9,801

CAC (Cost per customer):
  Ad spend: €1,500 / 33 clientes = ~€45 CAC
  
LTV (Lifetime Value):
  Año 1: €297
  Año 2: €297 × 25% = €74
  Total: €371 LTV
  
PAYBACK: €45 CAC / (€297 / 12) = 1.8 semanas
RATIO: LTV:CAC = 8.2:1 (muy saludable, >3 es bueno)
```

## COMPONENTES DE LA GARANTÍA

```
MENCIÓN 1: Landing page (final)
           "Sin riesgo, pruébalo"

MENCIÓN 2: Resultado page (sección dedicada)
           "30 días o devolución proporcional"

MENCIÓN 3: Checkout page (prominente)
           "Garantía visible: 30 días"

MENCIÓN 4: Email post-compra
           "Si no funciona, devolución sin preguntas"

MENCIÓN 5: Primera semana de protocolo
           "Recuerda: garantía hasta día 30"

┌─ Resultado: Cliente SABE sin dudas
│            que hay cero riesgo
│
└─ Efecto: +20-40% conversión
          (comparado con sin garantía)
```
