## 🎯 Casos de Uso Específicos

### "Tengo un nicho pero no sé si es viable"
```
LEER:
1. BLUEPRINT-REPLICABLE.md → Diagnóstico de Validez (15 min)
2. BLUEPRINT-REPLICABLE.md → Tus ejemplos más cercanos (10 min)

HACER:
1. Completa la tabla de 7 criterios
2. Si 6+ "Sí" → Procede
3. Si < 6 "Sí" → Reconsidera o ajusta el alcance
```

### "Tengo validación inicial y quiero hacer MVP"
```
LEER:
1. QUICK-START-48h.md → TODO (2-3 horas de lectura)

HACER:
1. Sigue el calendario día por día
2. Resultado: Landing + Quiz en 48h
3. Early validation con 50 leads
```

### "MVP está vivo, convertía mal, qué hago?"
```
LEER:
1. TEMPLATES-COPY.md → Patrones de headlines (15 min)
2. QUICK-START-48h.md → Troubleshooting (10 min)

HACER:
1. Reescribe hero usando patrón diferente
2. A/B test 2 headlines distintas
3. Medidas: mejor conversion o confirmación del problema
```

### "Tengo 100+ leads, algunos querían comprar, listo para escalar"
```
LEER:
1. BLUEPRINT-REPLICABLE.md → Guía Fase 3 + 4 + 5 (40 min)
2. BLUEPRINT-REPLICABLE.md → Métricas y KPIs (15 min)

HACER:
1. Setup analytics + email sequences completas
2. Lanzar soft launch a red
3. Escalar ads (si CAC < target)
4. Retention focus (seguimiento semanal, testimonios)
```
