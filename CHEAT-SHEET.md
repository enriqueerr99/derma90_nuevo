## THE FORMULA (Funciona en cualquier nicho)

```
PASO 1: VALIDAR PROBLEMA
├─ Existe un problema claro (sí/no)
├─ Solución actual es fragmentada (sí/no)
├─ Gente gasta $100+ intentando resolver (sí/no)
└─ Si 3 "sí" → Procede

PASO 2: DEFINIR SOLUCIÓN COMO SISTEMA
├─ No vendas producto (suplemento, app, etc)
├─ Vende PROTOCOLO (12 semanas)
├─ Incluye: producto + educación + seguimiento
└─ Precio: 5-10x el coste individual

PASO 3: CREAR FUNNEL CON QUIZ
├─ Landing (valida problema)
├─ Quiz 12Q (captura lead + data)
├─ Resultado personalizado (por perfil)
├─ Checkout sin fricción
└─ Confirmación + email

PASO 4: GARANTÍA VISIBLE
├─ Repite 5+ veces en el funnel
├─ Ofrece devolución o sesiones gratis
├─ Convierte conversión 20-40%
└─ Resultado: 3-5% final conversion

PASO 5: RETENCIÓN + REFUND RATE
├─ 85%+ completa el programa
├─ <5% devuelve dinero
├─ 20-30% recompra o upgradea
└─ Unit economics: profitable mes 1
```

## COPYWRITING RÁPIDO

```
HERO: [Validación del pain] + [Pregunta provocadora]
"Te miras al espejo y piensas: ¿quién es esta?"

SUBHEADLINE: [Problema tiene X raíces] + [Ningún producto aislado funciona]
"Porque perimenopausia tiene 3 puntos de falla simultáneos.
 Ningún producto aislado los cierra todos."

BODY SECTION 1: Normalizar frustración
"Llevas X años con este problema. Has gastado $Y.
 Nada funcionó porque..."

BODY SECTION 2: Educación + Framework
"El problema tiene 3 raíces:
 1. [Raíz 1]
 2. [Raíz 2]
 3. [Raíz 3]"

BODY SECTION 3: Transición a solución
"[Solución vieja] falla porque [limitación].
 [Nuestra solución] funciona porque [diferencial]."

GUARANTÍA: "Si en [tiempo] no ves [resultado], [oferta]"

CTA: [Acción específica] → [siguiente paso]
```

## PRICING LÓGICA

```
DERMA90: €297
├─ Suplemento retail: €20
├─ Educación: €50
├─ Coaching: €150
├─ Comunidad: €50
├─ Garantía: Invaluable
└─ DIFERENCIAL: Sistema coherente + responsabilidad

FÓRMULA:
Precio = (Gasto anual actual / 4-6 ) × (1 + premium por coaching)

Ejemplo: Si cliente gasta €1200/año en soluciones parciales
→ €1200 / 5 = €240 + 20% coaching = €288

Tu precio: €250-300 (psicológica: punto dulce es €297)
```

## TIMELINE 90 DÍAS

```
SEMANA 1-2: VALIDACIÓN
├─ Entrevistas (20+) sobre el problema
├─ Competitive analysis
├─ TAM/SAM estimate
└─ Definir 3 pilares

SEMANA 3-4: PRODUCT + COPY
├─ Oferta definida (duración, precio)
├─ Perfiles creados (3-4)
├─ Copy writing (usando templates)
└─ Protocolo diseñado

SEMANA 5-6: BUILD + TEST
├─ Landing page (Carrd/Webflow)
├─ Quiz (Typeform/Jotform)
├─ Resultado page (dinámico)
├─ Soft launch (50-100 personas)

SEMANA 7-8: ANALYZE + ITERATE
├─ Conversión analysis
├─ Feedback recolección
├─ Copy rewriting (A/B test)
├─ Product adjustments

SEMANA 9-10: PRE-LAUNCH
├─ Analytics setup
├─ Email sequences
├─ Fulfillment ready
├─ Legal docs

SEMANA 11-12: LAUNCH + SCALE
├─ Ads activation (primero $500)
├─ Daily optimization
├─ Retention focus
└─ Feedback loop

RESULTADO DÍA 90: Landing viva, 100-500 leads, 10-25 clientes pagadores
```

## ERRORES A EVITAR

```
❌ Vender PRODUCTO en lugar de SISTEMA
❌ Esperar al producto perfecto para lanzar
❌ Hacer landing sin validar el pain primero
❌ Pedir email antes de generar interés
❌ Esconder la garantía o mencionarla 1 sola vez
❌ Usar copy genérico o corporativo
❌ Ignorar el mobile (60% de tráfico)
❌ No responder emails rápido (trust builder)
❌ Pricing confuso o con decimales
❌ CTAs genéricos ("Más información" vs "Hacer el test")
```

## CUANDO LANZAR

```
✅ VERDE: Lanza YA
├─ Pain point validado (20+ entrevistas dicen "sí, eso soy yo")
├─ 3 pilares claros y específicos
├─ Copy que resonó en feedback
├─ Landing básico pero funcional
└─ Garantía clara

🟡 AMARILLO: Itera 1-2 semanas más
├─ Feedback mixto sobre el problema
├─ 3 pilares vagos o no tan claros
├─ Copy generado pero no testeado
├─ Landing bonito pero sin conversión
└─ Precio todavía no validado

❌ ROJO: No lances, reconsidera
├─ Nadie se identifica con el problema
├─ Solución fragmentada (pero no tanto)
├─ Copy muy técnico o corporativo
├─ No hay diferenciación clara
└─ Garantía débil o poco creíble
```

## LA VERDAD BRUTAL

```
Este modelo funciona porque:

1. Valida el problema antes de vender la solución
   (90% de marketeers hacen lo opuesto)

2. Define un SISTEMA, no un PRODUCTO
   (90% de startups venden productos aislados)

3. La garantía mata el risk perception
   (90% de gente nunca la menciona)

4. Personalización por perfil > Mensajería genérica
   (90% de landings hablan a todos, hablan a nadie)

5. Reputación > Hype
   (Honestidad vende más que promesas exageradas)

Resultado: 3-5% conversión (vs 0.5-1% industria)
Resultado: 85%+ retención (vs 30-50% industria)
Resultado: Unit economics positivos mes 1 (vs 6-12 meses)
```
