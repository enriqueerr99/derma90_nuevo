## Overview

The Gammapdf skill automates creating professional Gamma presentations that:
- Use the custom **Derma theme** (38utmdnlhlr39mm) consistently
- Are written in **Spanish**
- Incorporate **brand-accurate styling** (Terracota #B85C3C, Negro #12100e, Beige #faf8f5, Dorado #d4aa60, Gris #5a5550)
- Use **Playfair Display + DM Sans** typography
- Include **relevant images** from project assets when available
- Export to **PDF format** for distribution

## When to Use This Skill

Use Gammapdf whenever:
- Creating diagnostic documents for leads (BARRERA, DENSIDAD, ESTRÉS, SŌMA+ profiles)
- Generating educational or product materials
- Producing presentations that need professional branding
- Content needs to be in Spanish with IDERMA90 visual identity
- Output should be shareable as PDF

## How to Use This Skill

### Step 1: Understand the User's Content Request

The user will provide:
- **Content type** (diagnostic profile, product guide, educational material)
- **Topic/Product** (BARRERA, DENSIDAD, ESTRÉS, SŌMA+, etc.)
- **Key messages** or outline of what to cover
- **Target audience** (leads, customers, practitioners)

### Step 2: Create the Gamma Presentation

**Access Gamma.app:**
1. Open gamma.app in the browser
2. Click "Create" → "New Presentation"

**Configure the presentation:**
- **Theme**: Set to **Derma** (ID: 38utmdnlhlr39mm) — this is mandatory
- **Language**: Spanish (all text must be in Spanish)
- **Format**: presentation (slides)

**Structure the content:**
Use the diagnostic document structure when applicable:
1. **Portada/Hero** — Eye-catching title slide with brand colors
2. **Validación** — Customer success or problem identification
3. **Diagnosis** — Root cause analysis or condition assessment
4. **Ingredients/Solución** — Product features, benefits, or active ingredients
5. **Timeline** — Expected results, implementation timeline, or usage guide
6. **FAQs** — Common questions and answers

For other content types, organize logically with:
- Clear section breaks
- Visual hierarchy using the Derma theme
- Consistent brand colors (primary: Terracota #B85C3C)

**Add Images:**
1. Look for images in: `C:\Users\Enrique\OneDrive\Documentos\Claude\Projects\IDERMA\assets\images\`
2. Insert images that match the topic:
   - For ingredient profiles: individual ingredient images (vitamina-c.jpg, vitamina-a.jpg, etc.)
   - For BARRERA, DENSIDAD, ESTRÉS profiles: matching hero/background images
   - For SŌMA+: product and ingredient showcase images
3. Ensure images support the narrative and maintain visual consistency

### Step 3: Write Content in Spanish

- **Use professional Spanish** appropriate for skincare/dermatology
- **Maintain brand voice**: expert, scientific, empowering
- **Use visual metaphors** from the brand (barrera, densidad, estrés, regeneración)
- **Include data/science** when available (percentages, clinical results, active ingredient concentrations)
- **Call-to-action** on final slides when appropriate

### Step 4: Apply Brand Styling

The Derma theme handles most styling automatically. Ensure:
- **Title slides** use Terracota (#B85C3C) as primary color
- **Accent colors** include: Negro (#12100e), Beige (#faf8f5), Dorado (#d4aa60), Gris #5a5550
- **Typography** uses Playfair Display for headlines, DM Sans for body text
- **Spacing and layout** follow the theme's grid system
- **Visual consistency** across all slides

### Step 5: Export to PDF

Once the presentation is complete:
1. Click **Share** (top right)
2. Select **Export** → **PDF**
3. Configure PDF settings:
   - **Format**: Letter or A4 (user preference)
   - **Quality**: High
   - **Include speaker notes**: No (unless specified)
4. Download the PDF
5. Provide the PDF file to the user
